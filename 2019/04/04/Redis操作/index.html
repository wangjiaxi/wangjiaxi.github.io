Redis操作

发表于 2019-04-04 | 分类于 大数据

环境准备：centOS6.5

Redis是c语言开发的。
安装redis需要c语言的编译环境。如果没有gcc需要在线安装。

安装编译环境

yum install gcc-c++

安装步骤

  • 第一步：redis的源码(安装)包上传到linux系统。

  • 第二步：进入到压缩包所在的路径，解压缩redis。

  • 第三步：编译。进入redis源码目录（redis3.0.0）。

执行：make

  • 第四步：进入/usr/redis-3.0.0/src/安装。
